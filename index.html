Student Grade Calculator

A Robust Java CLI Tool for Academic Analytics

🚀 Overview

This project is a high-precision grade calculation system developed in Java. It is designed to handle multiple subjects, validate user inputs, and provide a detailed academic summary including total marks, average percentage, and final grade.

✨ Key Features

  • Defensive Programming: Validates marks (0-100) to prevent errors.
  • Buffer Safety: Uses advanced string parsing to handle Scanner inputs reliably.
  • Precision Math: Uses type-casting for accurate decimal percentages.
  • Clean UI: Formatted console output for professional result reporting.

💻 How to Run

Ensure you have Java JDK installed, then run these commands in your terminal:

javac StudentGradeCalculator.java
java StudentGradeCalculator
